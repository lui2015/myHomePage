# 个人工具汇总网页 — 需求文档

## 一、项目概述

| 项目 | 说明 |
|------|------|
| 项目名称 | 个人工具汇总网页（My Toolbox） |
| 项目类型 | 单页 Web 应用（SPA） |
| 目标用户 | 个人使用者 |
| 核心价值 | 将日常使用的在线工具/网站集中展示，一站式快捷访问与管理 |

---

## 二、核心功能需求

### 2.1 工具展示

- 以**卡片网格**布局展示所有已添加的工具，每张卡片包含：
  - 工具图标（自动抓取 favicon 或使用名称首字母占位）
  - 工具名称
  - 工具地址（可点击跳转，新标签页打开）
  - 操作按钮（编辑、删除）
- 支持**分类 / 标签**筛选（如：效率、开发、生活、学习等）
- 支持**关键词搜索**，实时过滤匹配的工具卡片

### 2.2 新增工具

- 页面提供「添加工具」按钮，点击弹出模态框（Modal）
- 表单字段：

| 字段 | 类型 | 必填 | 说明 |
|------|------|:----:|------|
| 工具名称 | 文本输入 | ✅ | 如"Google 翻译" |
| 工具地址 | URL 输入 | ✅ | 如 `https://translate.google.com` |
| 分类标签 | 下拉选择 / 可新建 | ❌ | 如"翻译"、"效率" |
| 工具描述 | 文本域 | ❌ | 简要说明工具用途 |

- 提交前进行**表单校验**（名称非空、地址格式合法）
- 提交成功后卡片实时出现在页面中

### 2.3 编辑工具

- 点击卡片上的「编辑」按钮，弹出与新增相同的模态框，预填当前信息
- 修改后保存，卡片内容实时更新

### 2.4 删除工具

- 点击卡片上的「删除」按钮，弹出**二次确认**对话框
- 确认后删除卡片，列表实时刷新

### 2.5 数据持久化

- 使用浏览器 `localStorage` 存储所有工具数据（JSON 格式）
- 页面刷新后数据不丢失
- 可选：提供「导出 JSON」和「导入 JSON」功能，方便备份与迁移

### 3.2 交互细节

| 交互场景 | 行为描述 |
|----------|----------|
| 点击卡片主体 | 新标签页打开工具地址 |
| 悬停卡片 | 卡片微上浮 + 阴影加深，显示编辑/删除按钮 |
| 点击「添加工具」 | 弹出模态框，含表单 |

| 点击分类标签 | 筛选对应分类下的工具 |
| 空状态 | 无工具时显示引导插图 + "添加你的第一个工具"按钮 |

## 四、UI 设计规范

### 4.1 整体风格

- **设计风格**：现代简约、毛玻璃（Glassmorphism）+ 渐变背景
- **色彩方案**：
  - 主色：`#6366F1`（靛蓝紫）
  - 辅色：`#8B5CF6`（紫色渐变）
  - 背景：浅灰渐变 `#F8FAFC` → `#E2E8F0`
  - 卡片背景：半透明白色 `rgba(255, 255, 255, 0.7)` + 模糊效果
  - 文字：主文字 `#1E293B`，辅助文字 `#64748B`
- **圆角**：卡片 `16px`，按钮 `8px`，输入框 `8px`
- **阴影**：`0 4px 16px rgba(0, 0, 0, 0.08)`

### 4.2 响应式适配

| 屏幕宽度 | 卡片列数 |
|----------|:--------:|
| ≥ 1200px | 4 列 |
| 768px – 1199px | 3 列 |
| 480px – 767px | 2 列 |
| < 480px | 1 列 |

### 4.3 动效

- 卡片出现：渐入 + 微上移（`fadeInUp`）
- 卡片删除：渐出 + 缩小（`fadeOutScale`）
- 模态框：淡入 + 从底部滑入
- 按钮悬停：缩放 `1.05` + 颜色过渡
